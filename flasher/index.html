⚡ FrankenChan

Flash your StackChan and its sensor nodes — right from this page.

1. What are you flashing?

Flashing the main robot firmware: face, voice chat, network monitor, IR remote, NFC, MQTT.

2. Plug it in and click

  1. Connect the device over USB-C
  2. Use Chrome or Edge on a desktop (Web Serial isn't in Safari or Firefox)
  3. Click below and pick the serial port that appears
This browser can't flash — use Chrome or Edge on desktop. Serial access isn't allowed on this page.

Port not showing up? Hold the reset/boot button while plugging in to force download mode.

3. First boot

  1. It opens a Wi-Fi network FrankenChan-xxxx (password stackchan)
  2. Join it — the setup page pops up (or browse to 192.168.4.1)
  3. Enter Wi-Fi, city and timezone. Optionally add an AI key for voice chat, and an MQTT broker for Home Assistant
  4. Save. Later it lives at frankenchan.local, with a dashboard at /dash

Controls: tap its back to talk · swipe to change screens · poke a cheek · pick it up, shake it, or knock on the desk.

3. First boot

  1. It opens a Wi-Fi network FC-Node-xxxx (password stackchan)
  2. Join it — the setup page appears
  3. Name the node, pick what it senses, and set its direction — StackChan turns its head that way when the sensor fires
  4. Save. It shows up on StackChan's dashboard automatically

No MAC address to copy and no channel to configure — nodes broadcast and sweep. The portal reopens for 30s on every boot if you want to change something.

Open firmware · the original M5Stack firmware can be restored any time with M5Burner.
